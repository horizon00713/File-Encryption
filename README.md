# 文件加密安全传输系统 

本项目是一个基于 C++ 实现的混合密码体制安全传输系统，旨在模拟并解决网络环境下的数据交互安全问题。

## 核心技术架构
系统底层从零原生实现了三大密码学核心算法：
* **AES-128 (CBC模式)**：负责大容量文件的高效、安全对称加密。
* **RSA (基于 NTL 库)**：负责非对称大整数运算、临时会话密钥的安全分发以及数字签名的生成。
* **SHA-1**：负责提取文件数据指纹，利用雪崩效应提供高灵敏度的完整性校验。

## 核心功能
本项目自带**交互式攻防防御终端**，完整实现了现代安全通信协议（如 HTTPS/TLS）的底层逻辑：
1. **PKI 基础设施建设**：模拟权威 CA 机构，为通信双方签发底层的数字证书，确立信任链。
2. **混合加密通信**：结合“数字信封”与“数字签名”技术，实现机密文件的端到端安全传输。
3. **主动防御测试**：系统内置防黑客演练模块，可精准识别并无情拦截“中间人假证书欺骗”以及“网络层密文篡改”等恶意攻击。
